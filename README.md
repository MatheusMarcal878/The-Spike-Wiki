🏐 The Spike Cross - Wiki Database
Uma enciclopédia interativa e centralizada para o jogo The Spike Cross. Este projeto permite que os usuários explorem personagens, consultem atributos detalhados, visualizem mecânicas de ascensão e gerenciem composições de times através de uma interface moderna e responsiva.

✨ Funcionalidades
Grade de Personagens Dinâmica: Carregamento automático de dados via JSON, organizados por posição (WS, MB, SE).

Sistema de Filtros Avançado: Filtre jogadores por nome, posição técnica ou Rank (S-, S, S+).

Modal de Detalhes Completo: * Visualização de atributos (ATQ, DEF, VEL, Salto).

Descrições de habilidades e frases icônicas.

Visualização de imagens de status e artes principais.

Simulador de Ascensão: Interface interativa para consultar bônus de melhoria do nível 0 ao 5.

Guia de Combinações: Exibição de sinergias entre personagens e benefícios de time.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica e organização dos modais.

CSS3: Design moderno com variáveis globais, layouts em Grid/Flexbox e efeitos de hover.

JavaScript (ES6+): Lógica de filtragem, manipulação assíncrona de dados (Fetch API) e gerenciamento de estado da interface.

JSON: Estrutura de dados centralizada para fácil manutenção e expansão da Wiki.
